## 📌 Project Overview

This project covers end-to-end manual testing of an e-commerce web application including modules like User Registration, Login, Product Search, Cart, Checkout, and Order Management.

The goal was to ensure the application functions correctly, performs well under normal usage, and meets business requirements through structured testing.

## ✅ Testing Types Performed

| Testing Type       | Description                                              |
|--------------------|----------------------------------------------------------|
| Functional Testing | Verified each feature works per business requirements    |
| Regression Testing | Re-tested after bug fixes to ensure no new issues        |
| Sanity Testing     | Quick checks after minor builds/changes                  |
| UI Testing         | Validated layout, alignment, and visual consistency      |
| API Testing        | Tested REST APIs using Postman (GET, POST)               |
| Database Testing   | Validated backend data using SQL queries                 |
